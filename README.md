# Dotfiles
---

This dotfile is my ArchLinux config.

## Description

VAIO S11 environment for ArchLinux.

### Login Manager
* SLiM

### Window Manager
* awesome

### Editor
* Vim
* NVim
* GVim

### Shell
* bash
* zsh

### Terminal
* ROXTerm

## Install
```
$ git clone https://github.com/Restoration/ArchLinux-dotfiles.git
$ cd ArchLinux-dotfiles/dotfiles
$ ./setup.sh
```
